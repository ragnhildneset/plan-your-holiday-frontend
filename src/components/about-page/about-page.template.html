About

Plan Your Holiday is a software project for the Course Software Engineering für betriebliche Anwendungen – Masterkurs: Web Application Engineering (IN2087) developed by team 59. Goal of the project is to design and implement a Single-Page Web Application utilizing the MEAN-Stack.



Purpose

The Website shall generate a schedule depending on the personal preferences of a user.

Utilized Software

Main Software-Bundel: MEAN-Stack (MongoDB, Express, AngularJS, Node.js). Development in Atom, Webstorm and Visual Studio. Version Control via GitHub.
